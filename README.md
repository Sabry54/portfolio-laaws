# Portfolio Theme - Instructions pour lancer le site

1. Installer WordPress localement (ex : Local by Flywheel, MAMP, XAMPP).
2. Copier ce thème dans `wp-content/themes/`.
3. Activer le thème via le tableau de bord WordPress.
4. Importer le contenu : `Outils → Importer → WordPress` et choisir `portfolio-content.xml`.
5. Télécharger tous les médias du dossier `media` dans `Médias → Ajouter`.
