Portfolio Theme - Instructions pour lancer le site

Installer WordPress localement
Utiliser Local by Flywheel, MAMP, XAMPP ou tout autre environnement local.

Ajouter le thème
Copier le contenu du dossier du repo (js, template-parts, footer.php, functions.php, header.php, index.php, style.css) dans wp-content/themes/portfolio-theme/.

Activer le thème
Aller dans le tableau de bord WordPress → Apparence → Thèmes → activer le thème Portfolio Theme.

Importer le contenu

Aller dans Outils → Importer → WordPress.

Choisir le fichier portfolio-laaws.WordPress.2025-11-11.xml (présent dans le repo).

Assigner le contenu à un utilisateur existant et cocher « Télécharger et importer les fichiers joints ».

Vérifier les médias
Tous les médias inclus dans le XML devraient être importés automatiquement.
Si certains fichiers manquent, les ajouter depuis le dossier media (si tu en as) via Médias → Ajouter.

Le site est prêt
Maintenant, ton site local devrait ressembler à celui que tu avais créé.
